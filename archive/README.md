# Research Archive

This directory contains archival research reports that provide the analytical background for the **Metaverse Research Landscape Evolution | 2007–2020** project.

## Archived Reports

### Metaverse Network Analysis: Trend, 2007–2022

This document preserves the R-based analytical workflow used to generate yearly semantic-network visualizations. It includes:

* annual network parameters and thresholds;
* emerging Metaverse-related concepts;
* yearly Top-10 concept tables;
* semantic-network visualizations from 2007 through 2022;
* the analytical code and workflow underlying the visual outputs.

### Metaverse Trend Report 2023

This report provides the broader research context for the project. It analyzes 128,858 Web of Science records associated with Metaverse-related concepts and examines:

* annual publication trends;
* country-level research distribution;
* disciplinary and research-field patterns;
* author-keyword frequencies;
* concept correlations;
* semantic-network structures;
* yearly changes in the Metaverse research landscape.

## Relationship to This Repository

The public visualization project focuses on the period **2007–2020** and is presented in two parts:

1. **2007–2016: The Virtual Worlds Era**
   Virtual worlds, simulation, education, rehabilitation, and virtual reality.

2. **2017–2020: The Infrastructure Transition**
   Blockchain, IoT, smart contracts, security, digital twins, machine learning, and artificial intelligence.

The archived reports provide the methodological and interpretive foundation for the scripts, figures, animations, and public videos contained in this repository.

## Data Availability

The repository includes research documentation, analytical scripts, aggregated outputs, figures, and videos.

The underlying Web of Science records and private intermediate research objects are not included because they are subject to database licensing and access restrictions.
