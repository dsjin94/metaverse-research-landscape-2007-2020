# Research Archive

This directory is reserved for local archival copies of the original analytical reports.

The original Metaverse research reports preserve the R-based analytical workflow, yearly parameters, semantic-network visualizations, emerging concepts, Top-10 concept tables, and broader Web of Science-based research analysis.

These archival reports are retained locally because they contain or depend on restricted Web of Science-derived data and private intermediate research objects.

The reports themselves are therefore not included in the public repository.

The public repository contains only:

* reproducible presentation scripts;
* derived yearly figures;
* final videos;
* project documentation.

The underlying Web of Science records, private intermediate objects, and complete archival reports are not published.
